What is Pentix?

This is an improved version of Tetris, where the number of squares in the figure is 5, i.e. is one more than in Tetris.

The idea of ​​the game is to place the figures most appropriately arriving from above in the playing field. In this case, if the line in the playing field is completely packed with squares, then it is cleared, and all that was topped is shifted downwards. Each figure consists of elementary squares interconnected by adjacent edges in a random way. The number of elementary squares in one figure equals 5. This is the fundamental difference between Pentix (5 - penta (lat.)) And the original Tetris (4 - tetra (lat.)), where the number of elementary squares constituting the figure is strictly 4. Figures arrive In a random sequence.

The goal of the game is to score the maximum possible number of points awarded for each successfully placed figure. The game ends when the game field is filled with figures.

In this game there are two modes: classic and lines. The idea of the first mode is that you need to play until the field is completely filled. The difficulty of the second mode is that you need to remove as fast as possible 20 lines.
